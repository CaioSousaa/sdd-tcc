- **Banco**: MongoDB/Mongoose — novo schema `Notification` (já modelado no context.md)

## Novos Arquivos

### Módulo Notification

**`src/modules/notification/port/notification-repository.port.ts`**
```typescript
export interface NotificationRepositoryPort {
  create(data: { owner: string; task: string; message: string }): Promise<void>;
  findByIdAndOwner(id: string, ownerId: string): Promise<NotificationRecord | null>;
  markAsRead(id: string): Promise<void>;
  listByOwner(ownerId: string): Promise<NotificationRecord[]>;
}
```

**`src/modules/notification/infra/repository/notification.repository.ts`**
- `create`: salva documento novo
- `findByIdAndOwner`: busca por `_id` + `owner`; retorna `null` se não encontrar
- `markAsRead`: `findByIdAndUpdate` setando `read: true`
- `listByOwner`: busca todos da owner, ordena por `createdAt: -1`, usa `.lean()`

**`src/modules/notification/infra/controllers/notification.controller.ts`**
Handlers:
- `markAsRead(req, res)`: chama `notificationService.markAsRead(req.params.id, req.userId!)`
- `list(req, res)`: chama `notificationService.listNotifications(req.userId!)`

**`src/modules/notification/factories/notification.factory.ts`**
Instancia `NotificationRepository` e `NotificationService`, exporta `notificationController` singleton.

**`src/routes/notification.routes.ts`**
```
PATCH /notifications/:id/read  → authenticate → notificationController.markAsRead
GET   /notifications            → authenticate → notificationController.list
```

const blacklist = new Set<string>();

export const isBlacklisted = (token: string) => blacklist.has(token);

### `src/modules/notification/services/notification.service.ts`
Erros customizados: `NotificationNotFoundError`, `NotificationAlreadyReadError`

- **`createFromAlert(taskId, ownerId)`**: busca a task; se não existir, retorna silenciosamente. Busca o user; se não existir, retorna silenciosamente. Cria notificação com `message = "Lembrete: a tarefa {task.title} está aguardando sua atenção"`.

## Tratamento de Erros

| Situação | Erro | HTTP | Mensagem |
|---|---|---|---|
| Notificação inexistente ou de outro usuário | `NotificationNotFoundError` | 404 | "Notificação não encontrada" |
| Notificação já lida | `NotificationAlreadyReadError` | 400 | "Notificação já marcada como lida" |
| Nenhum campo para atualizar | `NoFieldsToUpdateError` | 400 | "Nenhum campo para atualizar foi informado" |
| Senha fraca | `WeakPasswordError` | 400 | "A senha deve ter no mínimo 8 caracteres" |
| Token inválido/blacklisted | — (middleware) | 401 | mensagem padrão existente |
| Não autenticado | — (middleware) | 401 | mensagem padrão existente |
